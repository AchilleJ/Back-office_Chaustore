Voici comment sera divisé le code du back office

-un programme permettant de modifié/ajouté les données des tables "brand,category,size,color"
-un programme pour modifié/ajouté les données de la table "product"
-un programme pour modifié/ajouté les données de la table "stock"
-un programme pour supprimer une ligne dans les tables.

La visualisation des données de chaque tables sera lors de la modification.

la page d'acceuil possedera 6 boutons "brand gestion, category gestion etc.."
on clique sur un bouton et on lance un autre page avec modifié/ajouté/supprimer en fonction du bouton de gestion choisis
